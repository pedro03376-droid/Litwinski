Sincronização na Nuvem (Firebase)

Cole a URL do seu banco de dados Firebase Realtime Database para salvar os dados na nuvem e acessá-los de qualquer dispositivo.

📖 Como obter a URL do banco (passo a passo)
  1. Acesse console.firebase.google.com e abra seu projeto
  2. No menu lateral, vá em Build → Realtime Database
  3. Clique em "Criar banco de dados" (escolha Modo de teste)
  4. Copie a URL que aparece no topo, no formato https://seu-projeto-default-rtdb.firebaseio.com
  5. Cole a URL abaixo e clique em Conectar
